SGA - Security Lab

SGA - Security Lab Web güvenlik açıklarını test edebileceğiniz php ile kodlanmış web güvenlik zafiyetleri barındıran laboratuvar ortamıdır. Lab ortamını geliştirme, zafiyetlerin nasıl meydana geldiğini anlama ve önleme teknikleri SGA Security Lab üzerinden eğitim olarak işlenecektir.

KAZANIMLAR

  • Gerçekçi web uygulaması
  • Geliştiriciler için güvenlik bakış açısı kazanımı
  • Kritik güvenlik zafiyetlerini öğrenme
  • SQL Injection, XSS, CSRF, Command Injection, Session Fixation, File Inclusion, File Uplaod gibi OWASP TOP 10 zafiyetlerinin oluşumunu öğrenme ve engelleme
